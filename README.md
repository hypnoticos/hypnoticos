# HypnoticOS: 64-bit Hobby Operating System

## Prerequisites

You will need:
* an x86_64 cross-compiler (and other build tools, like `make`)
* grub-mkrescue

## Installing

To build:
```
make
```

To build and create an ISO:
```
make iso
```
